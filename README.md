# 문서 처리 도구

이 오픈소스 도구는 Hugging Face의 Transformers 및 기타 Python 라이브러리를 사용하여:

- **문서 번역** (한 언어에서 다른 언어로)
- **문서 요약**
- **고빈도 키워드 추출**

## 요구 사항
- Python 3.x
- 의존성 설치: `pip install -r requirements.txt`

## 사용법
1. 처리할 문서를 `docs/` 디렉토리에 넣습니다.
2. `python src/main.py` 명령어로 실행합니다.
